# Prerequisites

Before getting started, ensure your system has the following tools installed and running.

<Callout type="info">
  This tutorial was tested on **macOS (zsh)**. Commands should work on Linux too,
  but Docker setup steps may differ slightly.
</Callout>

## System Requirements

- macOS or Linux
- Terminal access (zsh or bash)

---

## Required Tools

### 1. Docker

Docker runs both the Gin application and Redis as containers on a shared network.

- [Install Docker Desktop for macOS →](https://www.docker.com/products/docker-desktop/)
- Make sure Docker Desktop is **open and running** before proceeding

Verify:

```bash
docker ps
```

<Callout type="warning">
  If Docker is not running, the app will fail to connect to Redis and Keploy commands
  will not work.
</Callout>

---

### 2. Keploy CLI


Keploy intercepts API traffic and converts it into replayable test cases — no test
writing required.

Install via:

```bash
curl --silent -O -L https://keploy.io/install.sh && source install.sh
```

Verify:

```bash
keploy --help
```

### 3. Go (Golang)

The sample application is written in Go. v1.20 or higher is recommended.

- [Install Go →](https://go.dev/dl/)

Verify:

```bash
go version
```

---

### 4. Git

Used to clone the sample repository.

Verify:

```bash
git --version
```

---

## Basic Knowledge

You don't need to be an expert, but familiarity with these will help:

- Making HTTP requests using `curl`
- Basic Docker concepts — containers, images, networks
- Running commands in a terminal
