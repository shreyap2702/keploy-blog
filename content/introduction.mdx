# Introduction

Keploy helps you test APIs with real traffic. Instead of writing large test suites from scratch, you record API interactions and replay them for consistent regression testing.

<Callout type="info">
This guide focuses on local developer workflows using Docker and a Node.js API.
</Callout>

## Why this workflow

- Reduce manual test-writing for common request/response flows
- Catch regressions before deployment
- Keep API behavior reproducible across environments

## What you will build

You will set up Keploy with Docker, record API calls, run replay tests, inspect failures, and fix regressions in a controlled loop.
