# Recording with Keploy

Record traffic while your API handles real requests.

## Start recording

```bash
keploy record -c "npm run dev"
```

Now hit your API using curl, Postman, or frontend flows.

## Captured artifacts

Keploy stores test cases and mocks that can be replayed later.
